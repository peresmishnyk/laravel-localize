AutomaticServiceProvider

AutomaticServiceProvider.php : 17

This trait automatically loads package stuff, if they're present in the expected directory. Stick to the conventions and your package will "just work". Feel free to override any of the methods below in your ServiceProvider if you need to change the paths.
